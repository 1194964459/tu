# 产品体验平台 MVP 技术方案

## 1. 项目概述

- **项目名称**：产品体验平台 (Product Demo Platform)
- **版本**：v1.3 (在线试用工作台)
- **技术栈**：Spring Boot + Vue 3 + MySQL
- **目标**：内部演示用的产品体验平台，支持多角色门户、AI 需求采集、智能推荐、在线试用

## 5. API 设计

### 5.1 用户模块
- `GET /api/users/{id}` 获取用户信息
- `GET /api/users/role/{role}` 按角色获取用户

### 5.2 产品模块
- `GET /api/products` 产品列表
- `GET /api/products/{id}` 产品详情
- `GET /api/products/recommend` 智能推荐

### 5.3 试用模块
- `POST /api/trials` 创建试用
- `GET /api/trials/{id}` 试用详情
- `POST /api/trials/{id}/feedback` 提交反馈
- `GET /api/trials/user/{userId}` 用户试用记录

### 5.4 AI 模块
- `POST /api/ai/chat` AI 对话
- `POST /api/ai/recommend` 方案推荐
